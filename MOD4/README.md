## Modification of the keyboard with diodes for each button

This PCB is designed in Proteus 7 for use with Cherry MX (5pin) keys. The circuit and size of the board corresponds to the keyboard of the PC "Вектор-06ц" and "ПК-6128ц".

* [KBDVektorG4.DSN](/MOD4/KBDVektorG4.DSN) -- circuit<br>
* [KBDVektorG4.LYT](/MOD4/KBDVektorG4.LYT) -- PCB layout<br>
* [KBDVektorG4 - CADCAM.ZIP](/MOD4/KBDVektorG4%20-%20CADCAM.ZIP) -- gerber file for production PCB
